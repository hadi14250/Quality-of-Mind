3. PERFORMANCE FIRST
   - Fast loading times
   - Optimized animations (GPU-accelerated)
   - Smooth scrolling without lag
   - Mobile-first responsive design

ANIMATION DETAILS
================================================================================

All animations follow modern best practices:

• Duration: 0.3s - 0.8s (feels instant, looks smooth)
• Easing: cubic-bezier(0.4, 0, 0.2, 1) - natural motion
• Properties: transform, opacity (GPU-friendly)
• Stagger: 0.1s - 0.15s delays for sequential reveals
• Hover: Transform scale/translate (no jarring effects)
• Mobile: Reduced motion where appropriate

ANIMATION TYPES USED:
---------------------
→ fadeInUp: Scroll-triggered element reveals
→ slideInLeft: Hero text entrance
→ scaleIn: Button and card reveals
→ float: Ambient background motion
→ pulse: Attention-drawing indicators
→ rotate: Spinning icons and borders
→ glow: Pulsing light effects
→ arrowBounce: Interactive feedback
